# gdmpasswd
Dumps the currently logged on users' password from gdm-password process memory.

```
root@pwnd:~# ./gdmpassword.sh
          __                                               __ 
.-----.--|  |.--------.-----.---.-.-----.-----.--.--.--.--|  |
|  _  |  _  ||        |  _  |  _  |__ --|__ --|  |  |  |  _  |
|___  |_____||__|__|__|   __|___._|_____|_____|________|_____|
|_____|               |__|                                    

USERNAME: root 
PASSWORD: r00tp@ss89!
```
